# Puzzle-Maker

A puzzle generator for:
 - Number Search
 - Crossword
 - Kakuro
 - Hidato
 - Nonogram
 - Cryptogram
 
 The stack is:
  - UI - Sciter (which uses HTML / CSS / TIscript(weird JS clone)
  - Puzzle Generator - Python
  - Puzzle Game - Pygame
  - Puzzle Image Generator - Pygame
  - PDF Generation - FPDF
  - PDF Tools (merging and splitting) - PyPDF2
